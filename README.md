- iOS 13.0+
- Xcode 12.3

- SnapKit 5.0.0

🗂 Resources

- AppDelegate
- SceneDelegate
- Assets
- APIServices 🗂
- Storyboards 🗂

🗂 Sources

  - ...

- Extensions 🗂		    
- Models 🗂

 ### Git Branch 전략

- main : 제품으로 출시될 수 있는 브랜치
- develop : 다음 출시 버전을 개발하는 브랜치
- feature : 기능을 개발하는 브랜치

[참고](https://woowabros.github.io/experience/2017/10/30/baemin-mobile-git-branch-strategy.html)
